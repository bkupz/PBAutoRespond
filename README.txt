README:
    What this program does is uses the open Pushbullet API to auto-respond to text messages on your behalf.

USEAGE:
    Change details in PushBullet.cfg to match your API Key, your device nickname, a VCF file of your contacts, and your own name in your contacts. More details can be found in the config file.
    Run Main.py to start the program and text your self on when you want to start the autoresponse. If you would like to change the default response text your self 'setmessage:' and then the message you would like.
